26 class RadialBasisFunctionsMethod(InterpolantBase):
27  """
28  Computes the trace of inverse of an invertible matrix :math:`\\mathbf{A} +
29  t \\mathbf{B}` using an interpolation scheme based on rational polynomial
30  functions (see details below).
31 
32  **Class Inheritance:**
33 
34  .. inheritance-diagram::
35  imate.InterpolateTraceinv.RadialBasisFunctionsMethod
36  :parts: 1
37 
38  :param A: Invertible matrix, can be either dense or sparse matrix.
39  :type A: numpy.ndarray
40 
41  :param B: Invertible matrix, can be either dense or sparse matrix.
42  :type B: numpy.ndarray
43 
44  :param options: A dictionary of input arguments for
45  :mod:`imate.traceinv.traceinv` module.
46  :type options: dict
47 
48  :param verbose: If ``True``, prints some information on the computation
49  process. Default is ``False``.
50  :type verbose: bool
51 
52  :param function_type: Can be ``1``, ``2``, or ``3``, which defines
53  different radial basis functions (see details below).
54  :type function_type: int
55 
56  **Interpolation Method**
57 
58  Define the function
59 
60  .. math::
61 
62  \\tau(t) = \\frac{\\mathrm{trace}\\left( (\\mathbf{A} +
63  t \\mathbf{B})^{-1} \\right)}{\\mathrm{trace}(\\mathbf{B}^{-1})}
64 
65  and :math:`\\tau_0 = \\tau(0)`. Then, we approximate :math:`\\tau(t)` by
66  radial basis functions as follows. Define
67 
68  .. math::
69 
70  x(t) = \\log t
71 
72  Depending whether ``function_type`` is set to ``1``, ``2``, or ``3``, one
73  of the following functions is defined:
74 
75  .. math::
76  :nowrap:
77 
78  \\begin{eqnarray}
79  y_1(t) &= \\frac{1}{\\tau(t)} - \\frac{1}{\\tau_0} - t, \\
80  y_2(t) &= \\frac{\\frac{1}{\\tau(t)}}{\\frac{1}{\\tau_0} + t} - 1, \\
81  y_3(t) &= 1 - \\tau(t) \\left( \\frac{1}{\\tau_0} + t \\right).
82  \\end{eqnarray}
83 
84  * The set of data :math:`(x, y_1(x))` are interpolated using
85  *cubic splines*.
86  * The set of data :math:`(x, y_2(x))` and :math:`(x, y_3(x))` are
87  interpolated using *Gaussian radial basis functions*.
88 
89  **Example**
90 
91  This class can be invoked from
92  :class:`imate.InterpolateTraceinv.InterpolateTraceinv` module using
93  ``method='RBF'`` argument.
94 
95  .. code-block:: python
96 
97  >>> from imate import generate_matrix
98  >>> from imate import InterpolateTraceinv
99 
100  >>> # Create a symmetric positive-definite matrix, size (20**2, 20**2)
101  >>> A = generate_matrix(size=20)
102 
103  >>> # Create an object that interpolates trace of inverse of A+tI
104  >>> # where I is identity matrix.
105  >>> TI = InterpolateTraceinv(A, method='RBF')
106 
107  >>> # Interpolate A+tI at some input point t
108  >>> t = 4e-1
109  >>> trace = TI.interpolate(t)
110  """
