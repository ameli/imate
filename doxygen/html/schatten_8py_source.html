33 
34  Parameters
35  ----------
36

47  :math:`\\Vert \\mathbf{A} \\Vert_p`. If `True`, it computes the
48  Schatten norm of the Gramian matrix, namely,
49  :math:`\\Vert \\mathbf{A}^{\\intercal} \\mathbf{A} \\Vert_p`.

63  default='exact'

65

67  * ``cholesky`` is only available for :math:`p \\leq 0`.
68  * ``hutchinson`` is only available for :math:`p < 0`.

86  * :ref:`imate.logdet.slq`

165  Notes
166  -----
167 
168  **Definition of the Norm:**
169 
170  This function defines the Schatten :math:`p`-norm of matrix
171  :math:`\\mathbf{A}` as
172 
173  .. math::
174  :label: schatten-eq-2
175 
176  \\Vert \\mathbf{A} \\Vert_p =
177  \\begin{cases}
178  \\left| \\mathrm{det}(\\mathbf{A})
179  \\right|^{\\frac{1}{n}}, & p=0, \\\\
180  \\left| \\frac{1}{n}
181  \\mathrm{trace}(\\mathbf{A}^{\\frac{1}{p}})
182  \\right|^{\\frac{1}{p}}, & p \\neq 0,
183  \\end{cases},
184 
185  where :math:`n` is the size of the matrix. When :math:`p \\geq 0`, the
186  above definition is the Schatten **norm**, and when :math:`p < 0`, the
187  above is the Schatten **anti-norm**.
188 
189  .. note::
190 
191  Conventionally, the Schatten norm is defined without the normalizing
192  factor :math:`\\frac{1}{n}` in :math:numref:`schatten-eq-2`. However,
193  this factor is justified by the continuity granted by
194 
195  .. math::
196  :label: schatten-continuous
197 
198  \\lim_{p \\to 0} \\Vert \\mathbf{A} \\Vert_p =
199  \\Vert \\mathbf{A} \\Vert_0.
200 
201  See [1]_ (Section 2) for details and the example below.
202 
203  **Method and Options Arguments:**
204 
205  The :func:`imate.schatten` calls the following functions:
206 
207  * If :math:`p > 0`, the computation is passed to :func:`imate.trace`
208  function.
209  * If :math:`p = 0`, the computation is passed to :func:`imate.logdet`
210  function.
211  * If :math:`p < 0`, the computation is passed to :func:`imate.traceinv`
212  function.
213 
214  The ``method`` and ``**options`` arguments to this functions are then
215  passed to the above functions directly. Hence, depending on the sign of
216  :math:`p`, see usage of the arguments in the documentation of the above
217  functions.
218 
219  References
220  ----------
221 
222  .. [1] Ameli, S., and Shadden. S. C. (2022). Interpolating Log-Determinant
223  and Trace of the Powers of Matrix :math:`\mathbf{A} + t \mathbf{B}`.
224  `arXiv: 2009.07385 <https://arxiv.org/abs/2207.08038>`_ [math.NA].

332 
333  Compute Schatten of a large sparse matrix using `SLQ` method for

335  :ref:`imate.trace.slq` function. Note that the SLQ method does not compute
336  norm exactly, rather, the result is an approximation using Monte-Carlo
337  sampling. The following example uses at least `100` samples.
338 
339  .. note::
340 
341  To see options passed to :func:`imate.schatten` for :math:`p > 0` in

343  function. In particular, note that the input matrix `A` should be
344  symmetric as a requirement to the `SLQ` method.
345 
346  .. code-block:: python
347  :emphasize-lines: 7, 8
348 
349  >>> # Generate a matrix of size one million
350  >>> from imate import toeplitz

352 
353  >>> # Approximate log-determinant using stochastic Lanczos quadrature

368  **Continuity of Norm in Order p:**
369 
370  Check the continuity of the Schatten norm over the order :math:`p`. Here,
371  the `eigenvalue` method is used as it can be applied to positive, negative,
372  and zero order :math:`p` among other methods.
373 
374  .. code-block:: python
375  :emphasize-lines: 20
376 
377  >>> # Importing packages
378  >>> import numpy
379  >>> import matplotlib.pyplot as plt
380  >>> import seaborn as sns
381  >>> from imate.sample_matrices import correlation_matrix
382  >>> from imate import schatten
383 
384  >>> # Plot settings (optional)
385  >>> sns.set(font_scale=1.15)
386  >>> sns.set_style("white")
387  >>> sns.set_style("ticks")
388 
389  >>> # Generate a sample matrix
390  >>> A = correlation_matrix(size=500)
391 
392  >>> # Compute norm over a range of order p, including p=0
393  >>> p = numpy.linspace(-10, 10, 201)
394  >>> norm = numpy.zeros_like(p)
395  >>> for i in range(p.size):
396  ... norm[i] = schatten(A, p=p[i], method='eigenvalue')
397 
398  >>> # Norm at p=0
399  >>> norm0 = schatten(A, p=0, method='eigenvalue')
400 
401  >>> # Plotting
402  >>> plt.semilogy(p, norm, color='black')
403  >>> plt.semilogy(0, norm0, 'o', color='black')
404  >>> plt.xlim([p[0], p[-1]])
405  >>> plt.ylim([1e-2, 1e2])
406  >>> plt.xlabel('$p$')
407  >>> plt.ylabel('$\\Vert \\mathbf{A} \\Vert_p$')
408  >>> plt.title(r'Schatten Norm')
409  >>> plt.show()
410 
411  .. image:: ../_static/images/plots/schatten_continuous.png
412  :align: center
413  :class: custom-dark
414  :width: 70%
415 
416  Since the Schatten norm in this function is defined as in
417  :math:numref:`schatten-eq-2`, the norm is continuous at :math:`p = 0`. See
418  :math:numref:`schatten-continuous`.
419  """
420 
421  if not isinstance(p, (int, numpy.integer, float)):
422  raise ValueError('"p" should be float or integer.')
423 
424  n = A.shape[0]
425 
426  if p == 0:
427  # Note, p in logdet is different than p in schatten.
428  output = logdet(A, gram=gram, p=1.0, return_info=return_info,
429  method=method, **options)
430 
431  if return_info:
432  logdet_ = output[0]
433  info = output[1]
434  else:
435  logdet_ = output
436 
437  schatten = numpy.exp(logdet_ / n)
438 
439  elif p > 0:
440  output = trace(A, gram=gram, p=p, return_info=return_info,
441  method=method, **options)
442 
443  if return_info:
444  trace_ = output[0]
445  info = output[1]
446  else:
447  trace_ = output
448 
449  schatten = (trace_ / n)**(1.0/p)
450 
451  elif p < 0:
452  output = traceinv(A, gram=gram, p=numpy.abs(p),
453  return_info=return_info, method=method, **options)
454 
455  if return_info:
456  traceinv_ = output[0]
457  info = output[1]
458  else:
459  traceinv_ = output
460 
461  schatten = (traceinv_ / n)**(1.0/p)
462 
463  if return_info:
464  return schatten, info
465  else:
466  return schatten
