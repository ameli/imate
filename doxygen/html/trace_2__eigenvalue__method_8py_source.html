31 
32 def eigenvalue_method(
33  A,
34  gram=False,
35  p=1.0,
36  return_info=False,
37  eigenvalues=None,
38  assume_matrix='gen',
39  non_zero_eig_fraction=0.9):
40  """
41  Trace of matrix using eigenvalue method.
42 
43  Given the matrix :math:`\\mathbf{A}` and the real exponent :math:`p`, the
44  following is computed:
45 
46  .. math::
47 
48  \\mathrm{trace} \\left(\\mathbf{A}^p \\right).
49 
50  If ``gram`` is `True`, then :math:`\\mathbf{A}` in the above is replaced by
51  the Gramian matrix :math:`\\mathbf{A}^{\\intercal} \\mathbf{A}`, and the
52  following is instead computed:
53 
54  .. math::
55 
56  \\mathrm{trace} \\left((\\mathbf{A}^{\\intercal}\\mathbf{A})^p
57  \\right).
58 
59  Parameters
60  ----------
61 
62  A : numpy.ndarray, scipy.sparse
63  A sparse or dense matrix.
64 
65  .. note::
66 
67  In the eigenvalue method, the matrix cannot be a type of
68  :class:`Matrix` or :class:`imate.AffineMatrixFunction` classes.
69 
70  gram : bool, default=False
71  If `True`, the trace of the Gramian matrix,
72  :math:`(\\mathbf{A}^{\\intercal}\\mathbf{A})^p`, is computed. The
73  Gramian matrix itself is not directly computed. If `False`, the
74  trace of :math:`\\mathbf{A}^p` is computed.
75 
76  p : float, default=1.0
77  The real exponent :math:`p` in :math:`\\mathbf{A}^p`.
78 
79  return_info : bool, default=False
80  If `True`, this function also returns a dictionary containing
81  information about the inner computation, such as process time,
82  algorithm settings, etc.
83 
84  eigenvalues : array_like [`float`], default=None
85  The array of all eigenvalues of `A`, if available. The size of this
86  array must be the same as the size of `A`. If `None`, the eigenvalues
87  of `A` are computed.
88 
89  assume_matrix : str {'gen', 'sym'}, default: 'gen'
90  Type of matrix. `gen` assumes generic matrix, while `sym` assumes
91  `A` is symmetric.
92 
93  non_zero_eig_fraction : float, default=0.9
94  A fraction (between `0` and `1`) of eigenvalues assumed to be non-zero.
95  For large matrices, it is not possible to compute all eigenvalues, and
96  only the largest eigenvalues can be computed and the rest are assumed
97  to be negligible. By setting this parameter, a fraction of
98  non-negligible eigenvalues is determined.
99 
100  Returns
101  -------
102 
103  trace : float or numpy.array
104  Trace of matrix.
105 
106  info : dict
107  (Only if ``return_info`` is `True`) A dictionary of information with
108  the following keys.
109 
110  * ``matrix``:
111  * ``data_type``: `str`, {`float32`, `float64`, `float128`}, type of
112  the matrix data.
113  * ``gram``: `bool`, whether the matrix `A` or its Gramian is
114  considered.
115  * ``exponent``: `float`, the exponent `p` in :math:`\\mathbf{A}^p`.
116  * ``assume_matrix``: `str`, {`gen`, `sym`}, determines whether
117  matrix is generic or symmetric.
118  * ``size``: `int`, the size of matrix `A`.
119  * ``sparse``: `bool`, whether the matrix `A` is sparse or dense.
120  * ``nnz``: `int`, if `A` is sparse, the number of non-zero elements
121  of `A`.
122  * ``density``: `float`, if `A` is sparse, the density of `A`, which
123  is the `nnz` divided by size squared.
124  * ``num_inquiries``: `int`, for the `eigenvalue` method, this is
125  always `1`.
126 
127  * ``device``:
128  * ``num_cpu_threads``: `int`, number of CPU threads used in shared
129  memory parallel processing.
130  * ``num_gpu_devices``: `int`, for the `eigenvalue` method, this is
131  always `0`.
132  * ``num_gpu_multiprocessors``: `int`, for the `eigenvalue` method,
133  this is always `0`.
134  * ``num_gpu_threads_per_multiprocessor``: `int`, for `eigenvalue`
135  method, this is always `0`.
136 
137  * ``time``:
138  * ``tot_wall_time``: `float`, total elapsed time of computation.
139  * ``alg_wall_time``: `float`, elapsed time of computation during
140  only the algorithm execution.
141  * ``cpu_proc_time``: `float`, the CPU processing time of
142  computation.
143 
144  * ``solver``:
145  * ``version``: `str`, version of imate.
146  * ``method``: 'eigenvalue'
147 
148  See Also
149  --------
150 
151  imate.logdet
152  imate.traceinv
153  imate.schatten
154 
155  Notes
156  -----
157 
158  **Computational Complexity:**
159 
160  The eigenvalue method uses spectral decomposition. The computational
161  complexity of this method is :math:`\\mathcal{O}(n^3)` where :math:`n` is
162  the matrix size. This method is only suitable for small matrices
163  (:math:`n < 2^{12}`). The solution is exact and can be used as a benchmark
164  to test randomized methods of computing trace.
165 
166  .. warning::
167 
168  It is not recommended to use this method for sparse matrices, as not
169  all eigenvalues of sparse matrices can be computed.
170 
171  Examples
172  --------
173 
174  **Dense matrix:**
175 
176  Compute the trace of :math:`\\mathbf{A}^{2.5}` for a symmetric Toeplitz
177  matrix created by :func:`imate.toeplitz` function:
178 
179  .. code-block:: python
180 
181  >>> # Import packages
182  >>> from imate import toeplitz, trace
183 
184  >>> # Create a symmetric matrix (setting gram=True makes it symmetric)
185  >>> A = toeplitz(2, 1, size=100, gram=True)
186 
187  >>> # Convert the sparse matrix to a dense matrix
188  >>> A = A.toarray()
189 
190  >>> # Compute trace with eigenvalue method
191  >>> trace(A, p=2.5, method='eigenvalue', assume_matrix='sym')
192  8849.423700390627
193 
194  Compute the trace of :math:`(\\mathbf{A}^{\\intercal} \\mathbf{A})^{2.5}`:
195 
196  .. code-block:: python
197 
198  >>> # Compute trace with eigenvalue method
199  >>> trace(A, gram=True, p=2.5, method='eigenvalue',
200  ... assume_matrix='sym')
201  1533618.9999999988
202 
203  **Precomputed Eigenvalues:**
204 
205  Alternatively, compute the eigenvalues of `A` in advance, and pass it to
206  this function:
207 
208  .. code-block:: python
209  :emphasize-lines: 6
210 
211  >>> # Compute eigenvalues of symmetric matrix A.
212  >>> from scipy.linalg import eigh
213  >>> eigenvalues = eigh(A, eigvals_only=True)
214 
215  >>> # Pass the eigenvalues to trace function
216  >>> trace(A, gram=True, p=2.5, method='eigenvalue',
217  ... eigenvalues=eigenvalues)
218  1533618.9999999988
219 
220  Pre-computing eigenvalues can be useful if :func:`imate.trace` function
221  should be called repeatedly for the same matrix `A` but other parameters
222  may change, such as `p`.
223 
224  **Print Information:**
225 
226  Print information about the inner computation:
227 
228  .. code-block:: python
229 
230  >>> tr, info = trace(A, method='eigenvalue', return_info=True)
231  >>> print(tr)
232  499.00000000000006
233 
234  >>> # Print dictionary neatly using pprint
235  >>> from pprint import pprint
236  >>> pprint(info)
237  {
238  'matrix': {
239  'assume_matrix': 'gen',
240  'data_type': b'float64',
241  'density': 1.0,
242  'exponent': 1.0,
243  'gram': False,
244  'nnz': 10000,
245  'num_inquiries': 1,
246  'size': 100,
247  'sparse': False
248  },
249  'solver': {
250  'method': 'eigenvalue',
251  'version': '0.15.0'
252  },
253  'device': {
254  'num_cpu_threads': 8,
255  'num_gpu_devices': 0,
256  'num_gpu_multiprocessors': 0,
257  'num_gpu_threads_per_multiprocessor': 0
258  },
259  'time': {
260  'alg_wall_time': 0.00996067700907588,
261  'cpu_proc_time': 0.01607515599999987,
262  'tot_wall_time': 0.00996067700907588
263  }
264  }
265 
266  **Sparse matrix:**
267 
268  Using a large matrix and ignoring 10% of its eigenvalues:
269 
270  .. code-block:: python
271 
272  >>> # Generate a symmetric sparse matrix
273  >>> A = toeplitz(2, 1, size=2000, gram=True)
274 
275  >>> # Assume only 80% of eigenvalues of A are non-zero
276  >>> trace(A, method='eigenvalue', assume_matrix='sym',
277  ... non_zero_eig_fraction=0.9)
278  9785.832766806298
279 
280  The above result is only an approximation since not all eigenvalues of `A`
281  are taken into account. To compare with the exact solution, use
282  :func:`imate.sample_matrices.toeplitz_trace` function.
283 
284  .. code-block:: python
285 
286  >>> from imate.sample_matrices import toeplitz_trace
287  >>> toeplitz_trace(2, 1, size=2000, gram=True)
288  9999
289 
290  There is a significant difference between the approximation with 90% of
291  eigenvalues and the actual solution. Because of this, it is not recommended
292  to use the eigenvalue method to compute trace.
293  """
294 
295  # Checking input arguments
296  check_arguments(A, eigenvalues, gram, p, assume_matrix,
297  non_zero_eig_fraction)
298 
299  init_tot_wall_time = time.perf_counter()
300  init_cpu_proc_time = time.process_time()
301 
302  if eigenvalues is None:
303 
304  # This is only relevant when A is sparse and we can only find a
305  # portion, bit not all, of its eigenvalues.
306  if p >= 0:
307  # when p is positive, the largest eigenvalues weight more
308  which_eigenvalues = 'LM'
309  else:
310  # when p is negative, the smallest eigenvalues weight more
311  which_eigenvalues = 'SM'
312 
313  # Compute eigenvalues of A (or A.T @ A is gram is True)
314  eigenvalues = compute_eigenvalues(A, gram, assume_matrix,
315  non_zero_eig_fraction,
316  which_eigenvalues)
317 
318  else:
319  # When eigenvalues of A are provided by the user
320  if gram:
321  eigenvalues = eigenvalues**2
322 
323  # Compute trace of inverse of matrix
324  not_nan = numpy.logical_not(numpy.isnan(eigenvalues))
325  trace = numpy.sum(eigenvalues[not_nan]**p)
326 
327  tot_wall_time = time.perf_counter() - init_tot_wall_time
328  cpu_proc_time = time.process_time() - init_cpu_proc_time
329 
330  # Dictionary of output info
331  info = {
332  'matrix':
333  {
334  'data_type': get_data_type_name(A),
335  'gram': gram,
336  'exponent': p,
337  'assume_matrix': assume_matrix,
338  'size': A.shape[0],
339  'sparse': isspmatrix(A),
340  'nnz': get_nnz(A),
341  'density': get_density(A),
342  'num_inquiries': 1
343  },
344  'device':
345  {
346  'num_cpu_threads': multiprocessing.cpu_count(),
347  'num_gpu_devices': 0,
348  'num_gpu_multiprocessors': 0,
349  'num_gpu_threads_per_multiprocessor': 0
350  },
351  'time':
352  {
353  'tot_wall_time': tot_wall_time,
354  'alg_wall_time': tot_wall_time,
355  'cpu_proc_time': cpu_proc_time,
356  },
357  'solver':
358  {
359  'version': __version__,
360  'method': 'eigenvalue',
361  }
362  }
363 
364  if return_info:
365  return trace, info
366  else:
367  return trace

374 def check_arguments(
375  A,
376  eigenvalues,
377  gram,
378  p,
379  assume_matrix,
380  non_zero_eig_fraction):
381  """
382  Checks the type and value of the parameters.
383  """
384 
385  # Check A
386  if (not isinstance(A, numpy.ndarray)) and (not scipy.sparse.issparse(A)):
387  raise TypeError('Input matrix should be either a "numpy.ndarray" or ' +
388  'a "scipy.sparse" matrix.')
389  elif A.shape[0] != A.shape[1]:
390  raise ValueError('Input matrix should be a square matrix.')
391 
392  # Check eigenvalues
393  if eigenvalues is not None:
394  if not isinstance(eigenvalues, numpy.ndarray):
395  raise TypeError('"eigenvalues" should be a numpy.ndarray.')
396  if eigenvalues.size != A.shape[0]:
397  raise ValueError('The length of "eigenvalues" does not match ' +
398  'the size of matrix "A".')
399 
400  # Check gram
401  if gram is None:
402  raise TypeError('"gram" cannot be None.')
403  elif not numpy.isscalar(gram):
404  raise TypeError('"gram" should be a scalar value.')
405  elif not isinstance(gram, bool):
406  raise TypeError('"gram" should be boolean.')
407 
408  # Check p
409  if p is None:
410  raise TypeError('"p" cannot be None.')
411  elif not numpy.isscalar(p):
412  raise TypeError('"p" should be a scalar value.')
413  elif not isinstance(p, (int, numpy.integer)):
414  TypeError('"p" cannot be an integer.')
415 
416  # Check assume_matrix
417  if assume_matrix is None:
418  raise TypeError('"assume_matrix" cannot be None.')
419  elif not isinstance(assume_matrix, str):
420  raise TypeError('"assume_matrix" should be a string.')
421  elif assume_matrix not in ['gen', 'sym']:
422  raise ValueError('"assume_matrix" should be either "gen" or "sym".')
423 
424  # Check non_zero_eig_fraction
425  if non_zero_eig_fraction is None:
426  raise TypeError('"non_zero_eig_fraction" cannot be None.')
427  elif not numpy.isscalar(non_zero_eig_fraction):
428  raise TypeError('"non_zero_eig_fraction" should be a scalar value.')
429  elif not isinstance(non_zero_eig_fraction, float):
430  raise TypeError('"non_zero_eig_fraction" should be a float type.')
431  elif non_zero_eig_fraction <= 0 or non_zero_eig_fraction >= 1.0:
432  raise ValueError('"non_zero_eig_fraction" should be greater then 0.0' +
433  'and smaller than 1.0.')

440 def compute_eigenvalues(
441  A,
442  gram,
443  assume_matrix,
444  non_zero_eig_fraction,
445  which_eigenvalues,
446  tol=1e-4):
447  """
448  """
449 
450  if gram:
451  # Gram matrix. Compute singular values of A.
452  if scipy.sparse.isspmatrix(A):
453 
454  # Sparse matrix
455  n = A.shape[0]
456  eigenvalues = numpy.empty(n)
457  eigenvalues[:] = numpy.nan
458 
459  # find 90% of eigenvalues, assume the rest are very close to zero.
460  num_none_zero_eig = int(n*non_zero_eig_fraction)
461 
462  singularvalues = \
463  scipy.sparse.linalg.svds(A, num_none_zero_eig,
464  which=which_eigenvalues,
465  return_singular_vectors=False,
466  tol=tol)
467  eigenvalues[:num_none_zero_eig] = singularvalues**2
468  else:
469 
470  # Dense matrix
471  if assume_matrix == 'sym':
472  singularvalues = numpy.linalg.svd(A, compute_uv=False,
473  hermitian=True)
474  else:
475  singularvalues = numpy.linalg.svd(A, compute_uv=False,
476  hermitian=False)
477  eigenvalues = singularvalues**2
478 
479  else:
480  # Not Gram matrix. Compute eigenvalues directly
481  if scipy.sparse.isspmatrix(A):
482 
483  # Sparse matrix
484  n = A.shape[0]
485  eigenvalues = numpy.empty(n)
486  eigenvalues[:] = numpy.nan
487 
488  # find 90% of eigenvalues, assume the rest are very close to zero.
489  num_none_zero_eig = int(n*non_zero_eig_fraction)
490 
491  if assume_matrix == 'sym':
492  eigenvalues[:num_none_zero_eig] = \
493  scipy.sparse.linalg.eigsh(A, num_none_zero_eig,
494  which=which_eigenvalues,
495  return_eigenvectors=False,
496  tol=tol)
497  else:
498  eigenvalues[:num_none_zero_eig] = \
499  scipy.sparse.linalg.eigs(A, num_none_zero_eig,
500  which=which_eigenvalues,
501  return_eigenvectors=False,
502  tol=tol)
503  else:
504 
505  # Dense matrix
506  if assume_matrix == 'sym':
507  eigenvalues = scipy.linalg.eigh(A, check_finite=False,
508  eigvals_only=True)
509  else:
510  eigenvalues = scipy.linalg.eig(A, check_finite=False)[0]
511 
512  return eigenvalues
