44  """
45  Interpolates the Schatten norm, or anti-norm, of an affine matrix function.
46 
47  Parameters
48  ----------
49 
50  A : numpy.ndarray, scipy.sparse matrix
51  A symmetric positive-definite matrix. Matrix can be dense or sparse.
52 
53  B : numpy.ndarray, scipy.sparse matrix, default=None
54  A symmetric positive-definite matrix. Matrix can be dense or sparse.
55  Matrix `B` should have the same size and type of matrix `A`. If `B`
56  is `None` (default value), it is assumed that `B` is an identity
57  matrix.
58 
59  p : float, default=2
60  The order :math:`p` is the Schatten :math:`p`-norm, which can be real
61  positive, negative or zero.
62 
63  ti : array_like(float), default=None
64  A list or an array of interpolant points that the interpolator use to
65  interpolate. If an empty list is give, ``[]``, a default list of

67 
68  kind : {`'EXT'`, `'EIG'`, `'MBF'`, `'IMBF'`, `'RBF'`, `'CRF'`, `'SPL'`, \
69  `'RPF'`}, default: `'IMBF'`
70  The algorithm of interpolation.
71 
72  :ref:`imate.InterpolateSchatten.ext`
73 
74  ``'EIG'`` Uses Eigenvalues of matrix exact
75  ``'MBF'`` Monomial Basis Functions interpolated
76  ``'IMBF'`` Inverse monomial basis functions interpolated
77  ``'RBF'`` Radial basis functions interpolated
78  ``'RPF'`` Rational polynomial functions interpolated
79 
80  verbose : bool, default=False
81  If `True`, it prints some information on the computation process.
82 
83  options : dict, default={``'method'``: ``'cholesky'``}
84  Options to pass to :func:`imate.schatten` function.
85 
86  kwargs : \\*\\*kwargs
87  Additional arguments to pass to each specific method. See documentation
88  for each ``kind``.
89 
90  Attributes
91  ----------
92  kind : str
93  Method of interpolation
94 
95  verbose : bool
96  Verbosity of the computation process
97 
98  n : int
99  Since of the matrix
100 
101  q : int
102  number of interpolant points
103 
104  p : float
105  Order of Schatten :math:`p`-norm

117  Notes
118  -----
119 
120  **Schatten Norm:**
121 
122  In this class, the Schatten :math:`p`-norm of matrix :math:`\\mathbf{A}` is
123  defined as
124 
125  .. math::
126  :label: schatten-eq-3
127 
128  \\Vert \\mathbf{A} \\Vert_p =
129  \\begin{cases}
130  \\left| \\mathrm{det}(\\mathbf{A})
131  \\right|^{\\frac{1}{n}}, & p=0, \\\\
132  \\left| \\frac{1}{n}
133  \\mathrm{trace}(\\mathbf{A}^{\\frac{1}{p}})
134  \\right|^{\\frac{1}{p}}, & p \\neq 0,
135  \\end{cases},
136 
137  where :math:`n` is the size of the matrix. When :math:`p \\geq 0`, the
138  above definition is the Schatten **norm**, and when :math:`p < 0`, the
139  above is the Schatten **anti-norm**.
140 
141  .. note::
142 
143  Conventionally, the Schatten norm is defined without the normalizing
144  factor :math:`\\frac{1}{n}` in :math:numref:`schatten-eq-3`. However,
145  this factor is justified by the continuity granted by
146 
147  .. math::
148  :label: schatten-continuous-3
149 
150  \\lim_{p \\to 0} \\Vert \\mathbf{A} \\Vert_p =
151  \\Vert \\mathbf{A} \\Vert_0.
152 
153  See [1]_ (Section 2) for details and the example below.
154 
155  **Interpolation of Affine Matrix Function:**
156 
157  This class interpolates the one-parameter matrix function:
158 
159  .. math::
160 
161  t \\mapsto \\| \\mathbf{A} + t \\mathbf{B} \\|_p,
162 
163  where the matrices :math:`\\mathbf{A}` and :math:`\\mathbf{B}` are
164  symmetric and positive semi-definite (positive-definite if :math:`p < 0`)
165  and :math:`t` is a real parameter.
166 
167  The interpolator is initialized by providing :math:`q` interpolant points
168  :math:`t_i`, :math:`i = 1, \\dots, q`, which are often logarithmically
169  spaced in some interval :math:`t_i \\in [t_1, t_p]`. The interpolator can
170  interpolate the above function at inquiry points :math:`t \\in [t_1, t_p]`
171  using various methods.
172 
173  References
174  ----------
175 
176  .. [1] Ameli, S., and Shadden. S. C. (2022). Interpolating Log-Determinant
177  and Trace of the Powers of Matrix :math:`\mathbf{A} + t \mathbf{B}`.
178  `arXiv: 2009.07385 <https://arxiv.org/abs/2207.08038>`_ [math.NA].
179 
180  Examples
181  --------
182 
183  Interpolate the trace of inverse of the affine matrix function
184  :math:`\\mathbf{A} + t \\mathbf{B}`:
185 
186  .. code-block:: python
187 
188  >>> # Generate two sample matrices (symmetric and positive-definite)
189  >>> from imate import generate_matrix
190  >>> A = generate_matrix(size=20, correlation_scale=1e-1)
191  >>> B = generate_matrix(size=20, correlation_scale=2e-2)
192 
193  >>> # Initialize interpolator object
194  >>> from imate import InterpolateSchatten
195  >>> ti = [1e-2, 1e-1, 1, 1e1]
196  >>> TI = InterpolateSchatten(A, B, ti)
197 
198  >>> # Interpolate at an inquiry point t = 0.4
199  >>> t = 4e-1
200  >>> Trace = TI.interpolate(t)
201 
202  Interpolate an array of inquiry points
203 
204  .. code-block:: python
205 
206  >>> # Interpolate at an array of points
207  >>> import numpy
208  >>> t_array = numoy.logspace(-2, 1, 10)
209  >>> Trace = TI.interpolator(t_array)
210 
211  By default, the interpolation kind is ``'IMBF'``. Use a different
212  interpolation kind, such as ``'RBF'`` by
213 
214  .. code-block:: python
215 
216  >>> TI = InterpolateSchatten(A, B, ti, kind='RBF')
217 
218  By default, the trace is computed with the Cholesky decomposition method as
219  the interpolant points. Configure the
220  computation method by ``options`` as
221 
222  .. code-block:: python
223 
224  >>> # Specify arguments to imate.omputeTraceOfInverse in a dictionary
225  >>> options = \
226  ... {
227  ... 'method': 'hutchinson',
228  ... 'NumIterations': 20
229  ... }
230 
231  >>> # Pass the options to the interpolator
232  >>> TI = InterpolateSchatten(A, B, ti,
233  >>> options=options)
234 
235  See Also
236  --------
237  traceinv : Computes trace of inverse of a matrix.
238

245  def __init__(self, A, B=None, p=2, ti=[], kind='IMBF', verbose=False,
246  options={'method': 'cholesky'}, **kwargs):
247  """
248  Initializes the object depending on the method.
249  """
250 
251  # Attributes
252  self.kindkind = kind
253  self.verboseverbose = verbose
254  self.nn = A.shape[0]
255  if ti is not None and not numpy.isscalar(ti):
256  self.qq = len(ti)
257  else:
258  self.qq = 0
259  self.pp = p
260 
261  # Define an interpolation object depending on the given method
262  if kind == 'EXT':
263  # Exact computation, not interpolation
264  self.interpolatorinterpolator = ExactMethod(
265  A, B, p=self.pp, options=options, verbose=verbose, **kwargs)
266 
267  elif kind == 'EIG':
268  # Eigenvalues kind
269  self.interpolatorinterpolator = EigenvaluesMethod(
270  A, B, p=self.pp, options=options, verbose=verbose, **kwargs)
271 
272  elif kind == 'MBF':
273  # Monomial Basis Functions kind
274  self.interpolatorinterpolator = MonomialBasisFunctionsMethod(
275  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
276  **kwargs)
277 
278  elif kind == 'IMBF':
279  # Inverse Monomial Basis Functions kind
280  self.interpolatorinterpolator = InverseMonomialBasisFunctionsMethod(
281  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
282  **kwargs)
283 
284  elif kind == 'RBF':
285  # Radial Basis Functions kind
286  self.interpolatorinterpolator = RadialBasisFunctionsMethod(
287  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
288  **kwargs)
289 
290  elif kind == 'CRF':
291  # Chebushev Rational Functions
292  self.interpolatorinterpolator = ChebyshevRationalFunctionsMethod(
293  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
294  **kwargs)
295 
296  elif kind == 'SPL':
297  # Spline
298  self.interpolatorinterpolator = SplineMethod(
299  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
300  **kwargs)
301 
302  elif kind == 'RPF':
303  # Rational Polynomial Functions kind
304  self.interpolatorinterpolator = RationalPolynomialFunctionsMethod(
305  A, B, p=self.pp, ti=ti, options=options, verbose=verbose,
306  **kwargs)
307 
308  else:
309  raise ValueError("'kind' is invalid. Select one of 'EXT', " +
310  "'EIG', 'MBF', 'IMBF', 'RBF', 'CRF', 'SPL', or " +
311  "'RPF'.")

328  def eval(self, t):
329  """
330  Computes the function :math:`\\mathrm{trace}\\left( (\\mathbf{A} +
331  t \\mathbf{B})^{-1} \\right)` at the input point :math:`t` using exact
332  method.
333 
334  The computation method used in this function is exact (no
335  interpolation). This function is primarily used to compute traceinv on
336  the *interpolant points*.
337 
338  Parameters
339  ----------
340  t : float or numpy.array
341  An inquiry point, which can be a single number, or an array of
342  numbers.
343 
344  Returns
345  -------
346  traceinv : float or numpy.array
347  The exact value of the traceinv function.
348  """
349 
350  if isinstance(t, Number):
351  # Single number
352  T = self.interpolatorinterpolator.eval(t)
353 
354  else:
355  # An array of points
356  T = numpy.empty((len(t), ), dtype=float)
357  for i in range(len(t)):
358  T[i] = self.interpolatorinterpolator.eval(t[i])
359 
360  return T

366  def _compare_with_exact_solution(self, t, schatten):
367  """
368  Computes the trace with exact method (no interpolation), then compares
369  it with the interpolated solution.
370 
371  Parameters
372  ----------
373  t : numpy.array
374  Inquiry points
375 
376  Trace: float or numpy.array
377  The interpolated computation of trace.
378 
379  Returns
380  -------
381  exact : float or numpy.array
382  Exact solution of trace.
383 
384  relative_error : float or numpy.array
385  Relative error of interpolated solution compared to the exact
386  solution.
387  """
388 
389  if self.kindkind == 'EXT':
390 
391  # The Trace results are already exact. No need to recompute again.
392  schatten_exact = schatten
393  schatten_relative_error = numpy.zeros(t.shape)
394 
395  else:
396 
397  # Compute exact solution
398  schatten_exact = self.evaleval(t)
399  schatten_relative_error = 1.0 - (schatten / schatten_exact)
400 
401  return schatten_exact, schatten_relative_error

407  def interpolate(self, t):
408  """
409  Interpolates :math:`\\mathrm{trace} \\left( (\\mathbf{A} +
410  t \\mathbf{B})^{-1} \\right)` at :math:`t`.
411 
412  Parameters
413  ----------
414  t : float, list, numpy.array
415  The inquiry point(s) to be interpolated.
416 
417  compare_with_exact : bool, default=False
418  If `True`, it computes the trace with exact solution, then compares
419  it with the interpolated solution. The return values of the
420  :func:`InterpolateSchatten.interpolate` function become
421  interpolated trace, exact solution, and relative error. **Note:**
422  When this option is enabled, the exact solution will be computed
423  for all inquiry points, which can take a very long time.
424 
425  plot : bool, default=False
426  If `True`, it plots the interpolated trace versus the inquiry
427  points. In addition, if the option `compare_with_exact` is also set
428  to `True`, the plotted diagram contains both interpolated and exact
429  solutions and the relative error of interpolated solution with
430  respect to the exact solution.
431 
432  Returns
433  -------
434  trace : float or numpy.array
435  The interpolated value of the traceinv function.
436 
437  Notes
438  -----
439 
440  **Plotting:** Regarding the plotting of the graph of interpolation:
441 
442  * If no graphical backend exists (such as running the code on a
443  remote server or manually disabling the X11 backend), the plot
444  will not be shown, rather, it will ve saved as an ``svg`` file in
445  the current directory.
446  * If the executable ``latex`` is on the path, the plot is rendered
447  using :math:`\\rm\\laTeX`, which then, it takes a bit
448  longer to produce the plot.
449  * If :math:`\\rm\\laTeX` is not installed, it uses any available
450  San-Serif font to render the plot.
451 
452  To manually disable interactive plot display, and save the plot as
453  ``SVG`` instead, add the following in the very beginning of your code
454  before importing ``imate``:
455 
456  .. code-block:: python
457 
458  >>> import os
459  >>> os.environ['IMATE_NO_DISPLAY'] = 'True'
460  """
461 
462  if isinstance(t, Number):
463  # Single number
464  schatten = self.interpolatorinterpolator.interpolate(t)
465 
466  else:
467  # An array of points
468  schatten = numpy.empty((len(t), ), dtype=float)
469  for i in range(len(t)):
470  schatten[i] = self.interpolatorinterpolator.interpolate(t[i])
471 
472  return schatten

478  def bound(self, t):
479  """
480  Lower bound of the function :math:`t \\mapsto \\mathrm{trace} \\left(

484 
485  .. math::
486 
487  \\mathrm{trace}((\\mathbf{A}+t\\mathbf{B})^{-1}) \\geq
488  \\frac{n^2}{\\mathrm{trace}(\\mathbf{A}) +
489  \\mathrm{trace}(t \\mathbf{B})}
490 
491  Parameters
492  ----------
493  t : float or numpy.array
494  An inquiry point or an array of inquiry points.
495 
496  Returns
497  -------
498  lb : float or numpy.array
499  Lower bound of the affine matrix function.
500  """
501 
502  if isinstance(t, Number):
503  # Single number
504  schatten_bound = self.interpolatorinterpolator.bound(t)
505 
506  else:
507  # An array of points
508  schatten_bound = numpy.empty((len(t), ), dtype=float)
509  for i in range(len(t)):
510  schatten_bound[i] = self.interpolatorinterpolator.bound(t[i])
511 
512  return schatten_bound

518  def upper_bound(self, t):
519  """
520  Upper bound of the function :math:`t \\mapsto \\mathrm{trace} \\left(
521  (\\mathbf{A} + t \\mathbf{B})^{-1} \\right)`.
522 
523  The upper bound is given by
524 
525  .. math::
526 
527  \\frac{1}{\\tau(t)} \\geq \\frac{1}{\\tau_0} + t
528 
529  where
530 
531  .. math::
532 
533  \\tau(t) = \\frac{\\mathrm{trace}\\left( (\\mathbf{A} +
534  t \\mathbf{B})^{-1}
535  \\right)}{\\mathrm{trace}(\\mathbf{B}^{-1})}
536 
537  and :math:`\\tau_0 = \\tau(0)`.
538 
539  Parameters
540  ----------
541  t : float or numpy.array
542  An inquiry point or an array of inquiry points.
543 
544  Returns
545  -------
546  ub : float or numpy.array
547  bound of the affine matrix function.
548  """
549 
550  if isinstance(t, Number):
551  # Single number
552  schatten_ub = self.interpolatorinterpolator.upper_bound(t)
553 
554  else:
555  # An array of points
556  schatten_ub = numpy.empty((len(t), ), dtype=float)
557  for i in range(len(t)):
558  schatten_ub[i] = self.interpolatorinterpolator.upper_bound(t[i])
559 
560  return schatten_ub
