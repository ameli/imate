imate.InterpolateSchatten(kind=’mbf’)#

class imate.InterpolateSchatten(A, B=None, p=0, options={}, verbose=False, ti=[], kind='mbf')

Computes the trace of inverse of an invertible matrix \(\mathbf{A} + t \mathbf{B}\) using

See also

This page describes only the mbf method. For other kinds, see imate.InterpolateSchatten().

\[\frac{1}{(\tau(t))^{q+1}} \approx \frac{1}{(\tau_0)^{q+1}} + \sum_{i=1}^{q+1} w_i t^i,\]

where

\[\tau(t) = \frac{\mathrm{trace}\left( (\mathbf{A} + t \mathbf{B})^{-1} \right)}{\mathrm{trace \left( \mathbf{B}^{-1} \right)}}\]

and \(\tau_0 = \tau(0)\) and \(w_{q+1} = 1\). To find the weight coefficient \(w_1\), the trace is computed at the given interpolant point \(t_1\) (see ti argument).

When \(q = 1\), meaning that there is only one interpolant point \(t_1\) with the function value \(\tau_1 = \tau(t_1)\), the weight coefficient \(w_1\) can be solved easily. In this case, the interpolation function becomes

\[\frac{1}{(\tau(t))^2} \approx \frac{1}{\tau_0^2} + t^2 + \left( \frac{1}{\tau_1^2} - \frac{1}{\tau_0^2} - t_1^2 \right) \frac{t}{t_1}.\]

Note

This class accepts only one interpolant point (\(q = 1\)). That is, the argument ti should be only one number or a list of the length 1.

Class Inheritance:

Parameters

  • A (numpy.ndarray) – Invertible matrix, can be either dense or sparse matrix.

  • B (numpy.ndarray) – Invertible matrix, can be either dense or sparse matrix.

  • options (dict) – A dictionary of input arguments for imate.traceinv.traceinv module.

  • verbose (bool) – If True, prints some information on the computation process. Default is False.

Example

This class can be invoked from imate.InterpolateTraceinv.InterpolateTraceinv module using method='MBF' argument.

>>> from imate import generate_matrix
>>> from imate import InterpolateTraceinv

>>> # Create a symmetric positive-definite matrix, size (20**2, 20**2)
>>> A = generate_matrix(size=20)

>>> # Create an object that interpolates trace of inverse of A+tI
>>> # where I is identity matrix.
>>> TI = InterpolateTraceinv(A, method='MBF')

>>> # Interpolate A+tI at some input point t
>>> t = 4e-1
>>> trace = TI.interpolate(t)

See also

This class can only accept one interpolant point. A better method is 'RMBF' which accepts arbitrary number of interpolant points. It is recommended to use the 'RMBF' (see imate.InterpolateTraceinv.RootMonomialBasisFunctionsMethod) instead of this class.
