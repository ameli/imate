25 class MonomialBasisFunctionsMethod(InterpolantBase):
26  """
27  Computes the trace of inverse of an invertible matrix :math:`\\mathbf{A} +
28  t \\mathbf{B}` using
29 
30  .. math::
31 
32  \\frac{1}{(\\tau(t))^{q+1}} \\approx \\frac{1}{(\\tau_0)^{q+1}} +
33  \\sum_{i=1}^{q+1} w_i t^i,
34 
35  where
36 
37  .. math::
38 
39  \\tau(t) = \\frac{\\mathrm{trace}\\left( (\\mathbf{A} +
40  t \\mathbf{B})^{-1} \\right)}{\\mathrm{trace \\left( \\mathbf{B}^{-1}
41  \\right)}}
42 
43  and :math:`\\tau_0 = \\tau(0)` and :math:`w_{q+1} = 1`.
44  To find the weight coefficient :math:`w_1`, the trace is computed at the
45  given interpolant point :math:`t_1` (see ``ti`` argument).
46 
47  When :math:`q = 1`, meaning that there is only one interpolant point
48  :math:`t_1` with the function value :math:`\\tau_1 = \\tau(t_1)`, the
49  weight coefficient :math:`w_1` can be solved easily. In this case, the
50  interpolation function becomes
51 
52  .. math::
53 
54 
55  \\frac{1}{(\\tau(t))^2} \\approx \\frac{1}{\\tau_0^2} + t^2 +
56  \\left( \\frac{1}{\\tau_1^2} - \\frac{1}{\\tau_0^2} - t_1^2 \\right)
57  \\frac{t}{t_1}.
58 
59  .. note::
60 
61  This class accepts only *one* interpolant point (:math:`q = 1`). That
62  is, the argument ``ti`` should be only one number or a
63  list of the length 1.
64

72  :type A: numpy.ndarray
73 
74  :param B: Invertible matrix, can be either dense or sparse matrix.
75  :type B: numpy.ndarray
76 
77  :param options: A dictionary of input arguments for
78  :mod:`imate.traceinv.traceinv` module.
79  :type options: dict

81  :param verbose: If ``True``, prints some information on the computation
82  process. Default is ``False``.

84 
85  :example:
86 
87  This class can be invoked from
88  :class:`imate.InterpolateTraceinv.InterpolateTraceinv` module
89  using ``method='MBF'`` argument.
90 
91  .. code-block:: python
92 
93  >>> from imate import generate_matrix
94  >>> from imate import InterpolateTraceinv
95 
96  >>> # Create a symmetric positive-definite matrix, size (20**2, 20**2)
97  >>> A = generate_matrix(size=20)
98 
99  >>> # Create an object that interpolates trace of inverse of A+tI
100  >>> # where I is identity matrix.
101  >>> TI = InterpolateTraceinv(A, method='MBF')
102 
103  >>> # Interpolate A+tI at some input point t
104  >>> t = 4e-1
105  >>> trace = TI.interpolate(t)
106 
107  .. seealso::

109  This class can only accept one interpolant point. A better method is

111  recommended to use the ``'RMBF'`` (see
112  :class:`imate.InterpolateTraceinv.RootMonomialBasisFunctionsMethod`)
