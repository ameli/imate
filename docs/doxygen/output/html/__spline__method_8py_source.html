27 class SplineMethod(InterpolantBase):
28  """
29  Computes the trace of inverse of an invertible matrix :math:`\\mathbf{A} +
30  t \\mathbf{B}` using an interpolation scheme based on rational polynomial
31  functions (see details below).
32 
33  **Class Inheritance:**
34 
35  .. inheritance-diagram::
36  imate.InterpolateTraceinv.RadialBasisFunctionsMethod
37  :parts: 1
38 
39  :param A: Invertible matrix, can be either dense or sparse matrix.
40  :type A: numpy.ndarray
41 
42  :param B: Invertible matrix, can be either dense or sparse matrix.
43  :type B: numpy.ndarray
44 
45  :param options: A dictionary of input arguments for
46  :mod:`imate.traceinv.traceinv` module.
47  :type options: dict
48 
49  :param verbose: If ``True``, prints some information on the computation
50  process. Default is ``False``.
51  :type verbose: bool
52 
53  :param function_type: Can be ``1``, ``2``, or ``3``, which defines
54  different radial basis functions (see details below).
55  :type function_type: int
56 
57  **Interpolation Method**
58 
59  Define the function
60 
61  .. math::
62 
63  \\tau(t) = \\frac{\\mathrm{trace}\\left( (\\mathbf{A} +
64  t \\mathbf{B})^{-1} \\right)}{\\mathrm{trace}(\\mathbf{B}^{-1})}
65 
66  and :math:`\\tau_0 = \\tau(0)`. Then, we approximate :math:`\\tau(t)` by
67  radial basis functions as follows. Define
68 
69  .. math::
70 
71  x(t) = \\log t
72 
73  Depending whether ``function_type`` is set to ``1``, ``2``, or ``3``, one
74  of the following functions is defined:
75 
76  .. math::
77  :nowrap:
78 
79  \\begin{eqnarray}
80  y_1(t) &= \\frac{1}{\\tau(t)} - \\frac{1}{\\tau_0} - t, \\
81  y_2(t) &= \\frac{\\frac{1}{\\tau(t)}}{\\frac{1}{\\tau_0} + t} - 1, \\
82  y_3(t) &= 1 - \\tau(t) \\left( \\frac{1}{\\tau_0} + t \\right).
83  \\end{eqnarray}
84 
85  * The set of data :math:`(x, y_1(x))` are interpolated using
86  *cubic splines*.
87  * The set of data :math:`(x, y_2(x))` and :math:`(x, y_3(x))` are
88  interpolated using *Gaussian radial basis functions*.
89 
90  **Example**
91 
92  This class can be invoked from
93  :class:`imate.InterpolateTraceinv.InterpolateTraceinv` module using
94  ``method='RBF'`` argument.
95 
96  .. code-block:: python
97 
98  >>> from imate import generate_matrix
99  >>> from imate import InterpolateTraceinv
100 
101  >>> # Create a symmetric positive-definite matrix, size (20**2, 20**2)
102  >>> A = generate_matrix(size=20)
103 
104  >>> # Create an object that interpolates trace of inverse of A+tI
105  >>> # where I is identity matrix.
106  >>> TI = InterpolateTraceinv(A, method='RBF')
107 
108  >>> # Interpolate A+tI at some input point t
109  >>> t = 4e-1
110  >>> trace = TI.interpolate(t)
111  """
