30 class InterpolateTrace(InterpolateSchatten):
31  """
32  Interpolates the trace of powers of affine matrix functions.
33 
34  This class interpolates the function
35 
36  .. math::
37 
38  t \\mapsto \\mathrm{trace} \\left( (\\mathbf{A} + t \\mathbf{B})^p
39  \\right),
40 
41  where the matrices :math:`\\mathbf{A}` and :math:`\\mathbf{B}` are
42  Hermitian and positive semi-definite (positive-definite if :math:`p < 0`)
43  and :math:`t` is a real parameter.
44 
45  The interpolator is initialized by providing :math:`q` interpolant points
46  :math:`t_i`, :math:`i = 1, \\dots, q`, which are often logarithmically
47  spaced in some interval :math:`t_i \\in [t_1, t_p]`.
48  The interpolator can interpolate the above function at inquiry points
49  :math:`t \\in [t_1, t_p]` using various methods.
50 
51  Parameters
52  ----------
53  A : numpy.ndarray, scipy.sparse matrix
54  A positive-definite matrix. Matrix can be dense or sparse.
55 
56  B : numpy.ndarray, scipy.sparse matrix, default=None
57  A positive-definite matrix. Matrix can be dense or sparse. If
58  `None` or not provided, it is assumed that `B` is an identity
59  matrix of the shape of `A`.
60 
61  p : float , default=0
62  Power of matrix.
63 
64  ti : list(float), numpy.array(float), default=None
65  A list or an array of points that the interpolator use to
66  interpolate. The trace of inverse is computed for the interpolant
67  points with exact method. If `None`, a default list of interpolant
68  points is used.
69 
70  method : str, default=`'IMBF'`
71  Algorithm of interpolation. See table below.
72 
73  ========== ================================ ============
74  `kind` Description Results
75  ========== ================================ ============
76  ``'EXT'`` Computes without interpolation exact
77  ``'EIG'`` Uses Eigenvalues of matrix exact
78  ``'MBF'`` Monomial Basis Functions interpolated
79  ``'IMBF'`` Inverse monomial basis functions interpolated
80  ``'RBF'`` Radial basis functions interpolated
81  ``'RPF'`` Rational polynomial functions interpolated
82  ========== ================================ ============
83 
84  options : dict, default={'method': 'cholesky'}
85  A dictionary of arguments to pass to :mod:`imate.traceinv` module.
86 
87  verbose : bool, default=False
88  If `True`, it prints some information on the computation process.
89 
90  kwargs : \\*\\*kwargs
91  Additional options to pass to each specific interpolator class.
92  See the sub-classes of :mod:`imate.InterpolantBase`.
93 
94  Attributes
95  ----------
96  kind : str
97  Method of interpolation
98  verbose : bool
99  Verbosity of the computation process.
100  n : int
101  Since of the matrix
102  q : int
103  number of interpolant points
104 
105  Methods
106  -------
107  __call__
108  eval
109  interpolate
110  lower_bound
111  upper_bound
112  plot

114  Examples
115  --------
116 
117  Interpolate the trace of inverse of the affine matrix function
118  :math:`\\mathbf{A} + t \\mathbf{B}`:
119 
120  .. code-block:: python
121 
122  >>> # Generate two sample matrices (symmetric and positive-definite)
123  >>> from imate import generate_matrix
124  >>> A = generate_matrix(size=20, correlation_scale=1e-1)
125  >>> B = generate_matrix(size=20, correlation_scale=2e-2)
126 
127  >>> # Initialize interpolator object
128  >>> from imate import InterpolateSchatten
129  >>> ti = [1e-2, 1e-1, 1, 1e1]
130  >>> TI = InterpolateSchatten(A, B, ti)
131 
132  >>> # Interpolate at an inquiry point t = 0.4
133  >>> t = 4e-1
134  >>> Trace = TI.interpolate(t)
135 
136  Interpolate an array of inquiry points
137 
138  .. code-block:: python
139 
140  >>> # Interpolate at an array of points
141  >>> import numpy
142  >>> t_array = numoy.logspace(-2, 1, 10)
143  >>> Trace = TI.interpolator(t_array)
144 
145  By default, the interpolation kind is ``'IMBF'``. Use a different
146  interpolation kind, such as ``'RBF'`` by
147 
148  .. code-block:: python
149 
150  >>> TI = InterpolateSchatten(A, B, ti, kind='RBF')
151 
152  By default, the trace is computed with the Cholesky decomposition method as
153  the interpolant points. Configure the
154  computation method by ``options`` as
155 
156  .. code-block:: python
157 
158  >>> # Specify arguments to imate.omputeTraceOfInverse in a dictionary
159  >>> options = \
160  ... {
161  ... 'method': 'hutchinson',
162  ... 'NumIterations': 20
163  ... }
164 
165  >>> # Pass the options to the interpolator
166  >>> TI = InterpolateSchatten(A, B, ti,
167  >>> options=options)
168 
169  See Also
170  --------
171  traceinv : Computes trace of inverse of a matrix.
172

194  # ==================
195  # schatten to tracep
196  # ==================
197 
198  def _schatten_to_tracep(self, schatten):
199  """
200  Converts Schatten anti-norm to trace of matrix to the power of p.
201  """
202 
203  tracep = schatten**self.pp * self.nn
204  return tracep
205 
206  # ==================
207  # tracep to schatten
208  # ==================
209 
210  def _tracep_to_schatten(self, tracep):
211  """
212  Converts tracep to Schatten anti-norm.
213  """
214 
215  schatten = (tracep / self.nn)**(1.0/self.pp)
216  return schatten

403  """
404  Upper bound of the function :math:`t \\mapsto \\mathrm{trace} \\left(

406
