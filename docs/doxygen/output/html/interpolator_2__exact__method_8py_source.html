22 class ExactMethod(InterpolantBase):
23  """
24  Computes the trace of inverse of an invertible matrix :math:`\\mathbf{A}
25  + t \\mathbf{B}` using exact method (no interpolation is performed).
26  This class does not accept interpolant points as the result is not
27  interpolated, rather, used as a benchmark to compare the exact versus the
28  interpolated solution of the other classes.
29 
30  **Class Inheritance:**
31 
32  .. inheritance-diagram:: imate.InterpolateTraceinv.ExactMethod
33  :parts: 1
34 
35  :param A: A positive-definite matrix. Matrix can be dense or sparse.
36  :type A: numpy.ndarray or scipy.sparse.csc_matrix
37 
38  :param B: A positive-definite matrix.
39  If ``None`` or not provided, it is assumed that ``B`` is an identity
40  matrix of the shape of ``A``.
41  :type B: numpy.ndarray or scipy.sparse.csc_matrix
42 
43  :param options: A dictionary of arguments to pass to
44  :mod:`imate.traceinv` module.
45  :type options: dict
46 
47  :param verbose: If ``True``, prints some information on the computation
48  process. Default is ``False``.
49  :type verbose: bool
50 
51  :example:
52 
53  This class can be invoked from
54  :class:`imate.InterpolateTraceinv.InterpolateTraceinv` module
55  using ``method='EXT'`` argument.
56 
57  .. code-block:: python
58 
59  >>> from imate import generate_matrix
60  >>> from imate import InterpolateTraceinv
61 
62  >>> # Create a symmetric positive-definite matrix, size (20**2, 20**2)
63  >>> A = generate_matrix(size=20)
64 
65  >>> # Create an object that interpolates trace of inverse of A+tI
66  >>> # where I is identity matrix
67  >>> TI = InterpolateTraceinv(A, method='EXT')
68 
69  >>> # Interpolate A+tI at some input point t
70  >>> t = 4e-1
71  >>> trace = TI.interpolate(t)
72 
73  .. seealso::
74 
75  The result of the ``EXT`` method is identical with the eigenvalue
76  method ``EIG``, which is given by
77  :class:`imate.InterpolateTraceinv.EigenvaluesMethod`.
78  """
