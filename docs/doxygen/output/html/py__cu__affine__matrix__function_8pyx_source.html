31 cdef class pycuAffineMatrixFunction(pycuLinearOperator):
32  """
33  Defines a linear operator that is an affine function of a single parameter.
34  Given two matrices :math:`\\mathbf{A}` and :math:`\\mathf{B}`, the linear
35  operator is defined by
36 
37  .. math::
38 
39  \\mathbf{A}(t) = \\mathbf{A} + t \\mathbf{B},
40 
41  where :math:`t \\in \\mathbb{R}` is a parameter.
42 
43  **Initializing Object:**
44 
45  The matrices :math:`\\mathbf{A}` and :math:`\\mathbf{B}` are given at the
46  initialization of the object. These matrices can be a dense matrix as 2D
47  numpy arrays, or sparse matrices of any format (CSR, CSC, etc) using scipy
48  sparse module.
49 
50  .. note::
51 
52  Initializing the linear operator requires python's GIL. Also, the
53  following examples should be used in a ``*.pyx`` file and should be
54  compiled as cython's extension module.
55 
56  In the following example, we create the object ``Aop`` based on
57  scipy.sparse matrices of CSR format. Note the format of the input matrices
58  can also be anything other than ``'csr'``, such as ``'csc'``.
59 
60  .. code-block:: python
61 
62  >>> # Use this script in a *.pyx file
63  >>> import scipy.sparse
64 
65  >>> # Create to random sparse matrices
66  >>> n, m = 1000
67  >>> A = scipy.sparse.random(n, m, format='csr')
68  >>> B = scipy.sparse.random(n, m, format='csr')
69 
70  >>> # Create linear operator object
71  >>> from imate.linear_operator cimport AffineMatrixFunction
72  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A, B)
73 
74 
75  The following is an example of defining the operator with dense matrices:
76 
77  .. code-block:: python
78 
79  >>> # Use this script in a *.pyx file
80  >>> import numpy
81 
82  >>> # Create to random sparse matrices
83  >>> n, m = 1000
84  >>> A = numpy.random.randn((n, m), dtype=float)
85  >>> B = numpy.random.randn((n, m), dtype=float)
86 
87  >>> # Create linear operator object
88  >>> from imate.linear_operator cimport AffineMatrixFunction
89  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A, B)
90 
91  If the matrix ``B`` is not given, or if it is ``None``, or if it is ``0``,
92  then the linear operator assumes ``B`` is zero matrix. For example:
93 
94  .. code-block:: python
95 
96  # Case 1: Not providing B
97  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A)
98 
99  # Case 2: Setting B to None
100  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A, None)
101 
102  # Case 3: Setting B to scalar zero
103  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A, 0)
104 
105  If the matrix ``B`` is set to the scalar ``1``, the linear operator assumes
106  that ``B`` is the identity matrix. For example:
107 
108  .. code-block:: python
109 
110  >>> cdef AffineMatrixFunction Aop = AffineMatrixFunction(A, 1)
111 
112  **Setting the Parameter:**
113 
114  The parameter :math:`t` is given to the object ``Aop`` at **runtime** using
115  :func:`set_parameters` function.
116 
117  .. note::
118 
119  Setting the parameter using :func:`set_parameter` does not require
120  python's GIL, hence, the parameter can be set in ``nogil`` environment,
121  if desired.
122 
123  .. code-block:: python
124 
125  >>> # Use this script in a *.pyx file
126  >>> cdef double t = 1.0
127 
128  >>> # nogil environment is optional
129  >>> with nogil:
130  ... Aop.set_parameters(&t)
131 
132  Note that a *pointer* to the parameter should be provided to the function.
133 
134  **Matrix-Vector Multiplications:**
135 
136  The linear operator can perform matrix vector multiplication using
137  :func:`dot` function and the matrix-vector multiplication with the
138  transposed matrix using :func:`transpose_dot` function.
139 
140  .. note::
141 
142  Matrix-vector multiplication using :func:`dot` and
143  :func:`transpose_dot` functions do not require python's GIL, hence,
144  they can be called in a ``nogil`` environment, if desired.
145 
146  .. code-block:: python
147 
148  >>> # Use this script in a *.pyx file
149  >>> # Create a vectors as cython's memoryview to numpy arrays
150  >>> import numpy
151  >>> cdef double[:] b = numpy.random.randn(m)
152  >>> cdef double[:] c = numpy.empty((n, 1), dtype=float)
153 
154  >>> # Perform product on vector b and store the product on vector c
155  >>> with nogil:
156  ... Aop.dot(&b[0], &c[0])
157 
158  >>> # Perform product using the transpose of the operator
159  >>> with nogil:
160  >>> Aop.transpose_dot(&b[0], &c[0])
161 
162  .. seealso::
163 
164  :class:`Matrix`
165  """
