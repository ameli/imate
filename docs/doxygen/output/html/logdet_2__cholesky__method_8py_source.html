36 
37 def cholesky_method(
38  A,
39  gram=False,
40  p=1.0,
41  return_info=False,
42  cholmod=None):
43  """
44  Log-determinant of non-singular matrix using Cholesky method.
45 
46  Given the matrix :math:`\\mathbf{A}` and the real exponent :math:`p`, the
47  following is computed:
48 
49  .. math::
50 
51  \\mathrm{logdet} \\left(\\mathbf{A}^p \\right) = p \\log_e \\vert
52  \\det (\\mathbf{A}) \\vert.
53 
54  If ``gram`` is `True`, then :math:`\\mathbf{A}` in the above is replaced by
55  the Gramian matrix :math:`\\mathbf{A}^{\\intercal} \\mathbf{A}`, and the
56  following is instead computed:
57 
58  .. math::
59 
60  \\mathrm{logdet} \\left((\\mathbf{A}^{\\intercal}\\mathbf{A})^p
61  \\right) = 2p \\log_e \\vert \\det (\\mathbf{A}) \\vert.
62 
63  Parameters
64  ----------
65 
66  A : numpy.ndarray, scipy.sparse
67  A positive-definite sparse or dense matrix.
68 
69  .. warning::
70 
71  This function does not pre-check whether the input matrix is
72  positive-definite.
73 
74  .. note::
75 
76  In the Cholesky method, the matrix cannot be a type of
77  :class:`Matrix` or :class:`imate.AffineMatrixFunction` classes.
78 
79  gram : bool, default=False
80  If `True`, the log-determinant of the Gramian matrix,
81  :math:`(\\mathbf{A}^{\\intercal}\\mathbf{A})^p`, is computed. The
82  Gramian matrix itself is not directly computed. If `False`, the
83  log-determinant of :math:`\\mathbf{A}^p` is computed.
84 
85  p : float, default=1.0
86  The real exponent :math:`p` in :math:`\\mathbf{A}^p`.
87 
88  return_info : bool, default=False
89  If `True`, this function also returns a dictionary containing
90  information about the inner computation, such as process time,
91  algorithm settings, etc.
92 
93  cholmod : bool, default=None
94  If set to `True`, it uses the `Cholmod` library from `scikit-sparse`
95  package to compute the Cholesky decomposition. If set to `False`, it
96  uses `scipy.sparse.cholesky` method. If set to `None`, first, it tries
97  to use Cholmod library, but if Cholmod is not available, then it uses
98  `scipy.sparse.cholesky` method.
99 
100  Returns
101  -------
102 
103  logdet : float or numpy.array
104  Log-determinant of `A`.
105 
106  info : dict
107  (Only if ``return_info`` is `True`) A dictionary of information with
108  the following keys.
109 
110  * ``matrix``:
111  * ``data_type``: `str`, {`float32`, `float64`, `float128`}. Type of
112  the matrix data.
113  * ``gram``: `bool`, whether the matrix `A` or its Gramian is
114  considered.
115  * ``exponent``: `float`, the exponent `p` in :math:`\\mathbf{A}^p`.
116  * ``size``: `int`, the size of matrix `A`.
117  * ``sparse``: `bool`, whether the matrix `A` is sparse or dense.
118  * ``nnz``: `int`, if `A` is sparse, the number of non-zero elements
119  of `A`.
120  * ``density``: `float`, if `A` is sparse, the density of `A`, which
121  is the `nnz` divided by size squared.
122  * ``num_inquiries``: `int`, for the `cholesky` method, this is
123  always `1`.
124 
125  * ``device``:
126  * ``num_cpu_threads``: `int`, number of CPU threads used in shared
127  memory parallel processing.
128  * ``num_gpu_devices``: `int`, for the `cholesky` method, this is
129  always `0`.
130  * ``num_gpu_multiprocessors``: `int`, for the `cholesky` method,
131  this is always `0`.
132  * ``num_gpu_threads_per_multiprocessor``: `int`, for the `cholesky`
133  method, this is always `0`.
134 
135  * ``time``:
136  * ``tot_wall_time``: `float`, total elapsed time of computation.
137  * ``alg_wall_time``: `float`, elapsed time of computation during
138  only the algorithm execution.
139  * ``cpu_proc_time``: `float`, CPU processing time of computation.
140 
141  * ``solver``:
142  * `cholmod_used`: `bool`, whether the Cholmod from SparseSuite
143  library was used.
144  * ``version``: `str`, version of imate.
145  * ``method``: 'cholesky'
146 
147  See Also
148  --------
149 
150  imate.trace
151  imate.traceinv
152  imate.schatten
153 
154  Notes
155  -----
156 
157  **Algorithm:**
158 
159  The log-determinant is computed from the Cholesky decomposition
160  :math:`\\mathbf{A} = \\mathbf{L} \\mathbf{L}^{\\intercal}` as
161 
162  .. math::
163 
164  \\log | \\mathbf{A} | =
165  2 \\mathrm{trace}( \\log \\mathrm{diag}(\\mathbf{L})).
166 
167  The result is exact (no approximation) and could be used as a benchmark to
168  test other methods.
169 
170  **Computational Complexity:**
171 
172  The computational complexity of this method is
173  :math:`\\mathcal{O}(\\frac{1}{3}n^3)` for dense matrices and
174  :math:`\\mathcal{O}(\\frac{\\rho}{3} n^2)` for sparse matrices where
175  :math:`n` is the matrix size and :math:`\\rho` is the sparse matrix
176  density.
177 
178  **Implementation:**
179 
180  This function is essentially a wrapper for the Cholesky function of the
181  `scipy` and `scikit-sparse` packages and is primarily used for testing and
182  comparison (benchmarking) against the randomized methods that are
183  implemented in this package. If ``cholmod`` is set to `True`, this function
184  uses the `Suite Sparse
185  <https://people.engr.tamu.edu/davis/suitesparse.html>`_ package to compute
186  the Cholesky decomposition.
187 
188  Examples
189  --------
190 
191  Compute the log-determinant of a sparse positive-definite Toeplitz matrix:
192 
193  .. code-block:: python
194 
195  >>> # Import packages
196  >>> from imate import toeplitz, logdet
197 
198  >>> # Generate a sample symmetric and positive-definite matrix
199  >>> A = toeplitz(2, 1, size=100, gram=True)
200 
201  >>> # Compute log-determinant with Cholesky method (default method)
202  >>> logdet(A, method='cholesky')
203  138.62943611198907
204 
205  Print information about the inner computation:
206 
207  .. code-block:: python
208 
209  >>> ld, info = logdet(A, method='cholesky', return_info=True)
210  >>> print(ld)
211  138.6294361119891
212 
213  >>> # Print dictionary neatly using pprint
214  >>> from pprint import pprint
215  >>> pprint(info)
216  {
217  'matrix': {
218  'data_type': b'float64',
219  'density': 0.0298,
220  'exponent': 1.0,
221  'gram': False,
222  'nnz': 298,
223  'num_inquiries': 1,
224  'size': 100,
225  'sparse': True
226  },
227  'solver': {
228  'cholmod_used': True,
229  'method': 'cholesky',
230  'version': '0.13.0'
231  },
232  'device': {
233  'num_cpu_threads': 8,
234  'num_gpu_devices': 0,
235  'num_gpu_multiprocessors': 0,
236  'num_gpu_threads_per_multiprocessor': 0
237  },
238  'time': {
239  'alg_wall_time': 0.0007234140066429973,
240  'cpu_proc_time': 0.0009358710000000325,
241  'tot_wall_time': 0.0007234140066429973
242  }
243  }
244  """
245 
246  # Check input arguments
247  check_arguments(A, gram, p, return_info, cholmod)
248 
249  # Determine to use Sparse
250  sparse = False
251  if scipy.sparse.isspmatrix(A):
252  sparse = True
253 
254  # Determine to use suitesparse or scipy.sparse to compute cholesky
255  if suitesparse_installed and cholmod is not False and sparse:
256  use_cholmod = True
257  else:
258  use_cholmod = False
259 
260  init_tot_wall_time = time.perf_counter()
261  init_cpu_proc_time = time.process_time()
262 
263  if p == 0:
264 
265  # determinant is 1. Logdet is zero
266  trace = 0.0
267 
268  else:
269 
270  # Compute logdet of A without the exponent
271  if sparse:
272 
273  # Sparse matrix
274  if use_cholmod:
275  # Use Suite Sparse
276  Factor = sk_cholesky(A)
277  trace = Factor.logdet()
278  else:
279  # Use scipy
280  diag_L = sparse_cholesky(
281  A, diagonal_only=True).astype(numpy.complex128)
282  logdet_L = numpy.real(numpy.sum(numpy.log(diag_L)))
283  trace = 2.0*logdet_L
284 
285  else:
286 
287  # Dense matrix. Use scipy
288  L = scipy.linalg.cholesky(A, lower=True)
289  diag_L = numpy.diag(L).astype(numpy.complex128)
290  logdet_L = numpy.real(numpy.sum(numpy.log(diag_L)))
291  trace = 2.0*logdet_L
292 
293  # Taking into account of the exponent p
294  trace = trace*p
295 
296  # Gramian matrix
297  if gram:
298  trace = 2.0 * trace
299 
300  tot_wall_time = time.perf_counter() - init_tot_wall_time
301  cpu_proc_time = time.process_time() - init_cpu_proc_time
302 
303  # Dictionary of output info
304  info = {
305  'matrix':
306  {
307  'data_type': get_data_type_name(A),
308  'gram': gram,
309  'exponent': p,
310  'size': A.shape[0],
311  'sparse': isspmatrix(A),
312  'nnz': get_nnz(A),
313  'density': get_density(A),
314  'num_inquiries': 1
315  },
316  'device':
317  {
318  'num_cpu_threads': multiprocessing.cpu_count(),
319  'num_gpu_devices': 0,
320  'num_gpu_multiprocessors': 0,
321  'num_gpu_threads_per_multiprocessor': 0
322  },
323  'time':
324  {
325  'tot_wall_time': tot_wall_time,
326  'alg_wall_time': tot_wall_time,
327  'cpu_proc_time': cpu_proc_time,
328  },
329  'solver':
330  {
331  'version': __version__,
332  'method': 'cholesky',
333  'cholmod_used': use_cholmod
334  }
335  }
336 
337  if return_info:
338  return trace, info
339  else:
340  return trace

347 def check_arguments(
348  A,
349  gram,
350  p,
351  return_info,

353  """
354  Checks the type and value of the parameters.
355  """
356 
357  # Check A
358  if (not isinstance(A, numpy.ndarray)) and (not scipy.sparse.issparse(A)):
359  raise TypeError('Input matrix should be either a "numpy.ndarray" or ' +
360  'a "scipy.sparse" matrix.')
361  elif A.shape[0] != A.shape[1]:
362  raise ValueError('Input matrix should be a square matrix.')
363 
364  # Check gram
365  if gram is None:
366  raise TypeError('"gram" cannot be None.')
367  elif not numpy.isscalar(gram):
368  raise TypeError('"gram" should be a scalar value.')
369  elif not isinstance(gram, bool):
370  raise TypeError('"gram" should be boolean.')
371 
372  # Check p
373  if p is None:
374  raise TypeError('"p" cannot be None.')
375  elif not numpy.isscalar(p):
376  raise TypeError('"p" should be a scalar value.')
377  elif isinstance(p, complex):
378  TypeError('"p" cannot be an integer or a float number.')
379 
380  # Check return info
381  if not isinstance(return_info, bool):
382  raise TypeError('"return_info" should be boolean.')
383 
384  # Check cholmod
385  if cholmod is not None:
386  if not isinstance(cholmod, bool):
387  raise TypeError('"cholmod" should be either "None", or boolean.')
388  elif cholmod is True and suitesparse_installed is False:
389  print(cholmod)
390  print(suitesparse_installed)
391  raise RuntimeError('"cholmod" method is not available. Either ' +
392  'install "scikit-sparse" package, or set ' +
393  '"cholmod" to "False" or "None".')
