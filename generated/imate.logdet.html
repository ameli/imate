imate.logdet¶

imate.logdet(A, method='cholesky', **options)[source]¶

Computes the log-determinant of full-rank matrix.

Parameters
Anumpy.ndarray, scipy.sparse matrix

An invertible matrix.

options**kwargs

Options for either of the methods.

Notes

For computing the trace of inverse with the stochastic Lanczos quadrature method (see imate.traceinv.slq_method), the preferred algorithm is the Lanczos tri-diagonalization, as opposed to Golub-Kahn bi-diagonalization.

In contrast to the above, the preferred slq method for computing log-determinant is the Golub-Kahn bi-diagonalization. The reason is that if the matrix \(\mathbf{A}\) has many singular values close to zero, bi-diagonalization performs better, and this matters when we compute determinant.

Examples

>>> # Import packages
>>> from imate import logdet
>>> from imate import generate_matrix

>>> # Generate a sample matrix
>>> A = generate_matrix(size=20, dimension=2)

>>> # Compute log-determinant with Cholesky method (default method)
>>> logdet_1 = logdet(A)

>>> # Compute log-determinant with stochastic Lanczos quadrature method
>>> # using Lanczos tridiagonalization
>>> logdet_2 = logdet(A, method='slq', num_iterations=20,
...     lanczosdegree=20, symmetric=True)

>>> # Compute log-determinant with stochastic Lanczos quadrature method
>>> # using Golub-Khan bi-diagonalization
>>> logdet_3 = logdet(A, method='slq', num_iterations=20,
...     lanczos_degree=20, symmetric=False)
